## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jeiku/registration-system.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Add your MongoDB URI in 'config/keys.js'

4. Start server
   ```sh
   npm start
   ```
